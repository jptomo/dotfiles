=============================
This is my dotfiles.
=============================

HOW TO USE
=============================

1. clone

::

   $ git clone git://github.com/jptomo/dotfiles.git somewhere

2. cd somewhere

::

   $ cd somewhere

3. add permission setup.sh to run

::

   $ chmod 0755 setup.sh

4. run setup.sh

::

   $ ./setup.sh

5. write own github account

   bellow file set github accounts.

::

   $ vim $HOME/.github.accounts
